# System engineering & Devops

Module of **System engineering** and **Devops**, carried out during **Full Stack Software Engineering studies** at **[ALX Software Engineer](https://www.alxafrica.com/)**.

## Technologies
* Scripts written in Bash 5.0.17
* Tested on Ubuntu 20.04.4 LTS

## Projects
All of the following folders are projects done during the studies:

| Project name | Description |
| ------------ | ----------- |
| [`0x00-shell_basics`](https://github.com/OusStack/alx-system_engineering-devops/tree/master/0x00-shell_basics) | It aims to learn about basics commands, navigation, files and directories in **Shell** |
| [`0x01-shell_permissions`](https://github.com/OusStack/alx-system_engineering-devops/tree/master/0x01-shell_permissions) | It aims to learn about man pages, permissions (owner, group and other) of files and directories in **Shell** |
| [`0x02-shell_redirections`](https://github.com/OusStack/alx-system_engineering-devops/tree/master/0x02-shell_redirections) | It aims to learn about how to handle standard input and output and how to combine commands and filters with redirections in **Shell** |
